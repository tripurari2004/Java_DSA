# Java DSA Learning Repository

Personal Java learning repository - practice programs for learning Data Structures and Algorithms (DSA) and core Java concepts.

## 📚 Contents

### 1. **Basic.java**
Introduction to Java fundamentals including:
- Basic syntax and structure
- `System.out.println()` for output
- Arithmetic operations
- Variable declaration and usage
- Simple calculations

**Key Concepts:**
- Hello World program
- Integer arithmetic
- Boolean values
- Variable assignment

---

### 2. **DataTypes.java**
Comprehensive guide to Java data types including:
- **Primitive Data Types:** int, double, char, boolean, float, long, byte, short
- **Non-Primitive Data Types:** String, Arrays
- **Type Casting:** Implicit and Explicit casting

**Key Concepts:**
- All 8 primitive data types with examples
- String and Array usage
- Type conversion (implicit and explicit)
- Enhanced for loop

---

### 3. **Binary.java**
Understanding binary number system and conversions:
- Converting decimal numbers to binary representation
- Binary arithmetic operations
- Bitwise operations basics
- Number system fundamentals

**Key Concepts:**
- Decimal to binary conversion
- Understanding binary representation
- Binary string display

---

### 4. **Operator.java**
Comprehensive guide to all Java operators:
- **Arithmetic Operators:** +, -, *, /, %
- **Assignment Operators:** +=, -=, *=, /=, %=
- **Relational Operators:** ==, !=, >, <, >=, <=
- **Logical Operators:** &&, ||, !
- **Bitwise Operators:** &, |, ^, ~
- **Shift Operators:** <<, >>, >>>
- **Ternary Operator:** ? :
- **Increment/Decrement:** ++, --

**Key Concepts:**
- Operator precedence and associativity
- Bitwise operations
- Logical evaluation

---

### 5. **UserInput.java**
Reading and processing user input:
- Using `Scanner` class for input
- Reading String, int, and double values
- Processing user input
- Resource management (closing Scanner)

**Key Concepts:**
- Scanner initialization
- Multiple input types
- Input validation concept

---

### 6. **Conditional.java**
Control flow using conditional statements:
- **if-else statements:** Simple, nested conditions
- **Switch statement:** Multiple case handling
- **Ternary operator:** Conditional assignment
- **Logical operators in conditions:** &&, ||, !
- **Nested if statements:** Complex decision making (e.g., leap year logic)

**Key Concepts:**
- Decision making
- Nested conditions
- Default cases
- Logical operations in conditionals

---

### 7. **WhileLoop.java**
Loop control using while loops:
- Simple while loop iteration

- Sum calculation of natural numbers
- User input with loops
- Loop counter management

**Key Concepts:**
- Loop initialization
- Loop condition

---

### 8. **ForLoop.java**
Loop control using for loops:
- Simple for loop iteration

**Key Concepts:**
- For loop structure (init, condition, increment)
- Nested loops

## 🚀 Getting Started

### Prerequisites
- Java Development Kit (JDK) installed (version 8 or higher)
- Any Java IDE (IntelliJ IDEA, Eclipse, VS Code) or command line

### Running the Programs

**Using Command Line:**
```bash
# Compile all files
javac *.java

# Run individual programs
java Basic
java DataTypes
java Binary
java Operator
java UserInput 
java Conditiona
java WhileLoop 
java ForLoop   
```

**Using IDE:**
- Open the file in your IDE
- Click the Run button or press Ctrl+F5

## 📖 Recommended Learning Path

2. **DataTypes.java** - Learn about data types and type casting
3. **Binary.java** - Understand binary system
